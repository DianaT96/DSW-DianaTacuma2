Propósito del sitio

Este sitio web trata sobre La interfaz de usuario, Interacción Humano Ordenador, el cual incluye los conceptos relacionados con el proceso de interacción de los usuarios y el ordenador, además resalta la importancia de producir un espacio adecuado para que el cliente controle su equipo con facilidad incluyendo elementos como menús, ventanas, contenido gráfico, etc.

La interfaz de usuario

Es el conjunto de los controles y canales sensoriales mediante los cuales un usuario puede comunicarse con una máquina, ejemplo: la pantalla, el teclado, las bocinas, Etc.

Su objetivo es permitir el funcionamiento y control efectivo de la máquina desde la interacción con el ser humano, transmitiendo la información de manera precisa para evitar que el usuario cometa errores durante la interacción y entregandole la información que necesita y pide.

Interacción Humano Ordenador

Es la disciplina que estudia cómo las personas interactúan con las computadoras y hasta qué punto las computadoras se desarrollan para interactuar con las personas, consta de tres componentes: los usuarios, los ordenadores y la interacción entre ellos.

Consiste en crear sistemas y marcos que mejoren la vida de las personas, Partiendo de sus necesidades se desarrollan herramientas y técnicas para construir sistemas adecuados que satisfagan dichas necesidades.
