Propósito del sitio

La finalidad es abordar el tema de las etiquetas semánticas las cuales nos permiten definir la estructura de nuestro documento indexandolas por los buscadores, es importante mencionar que cada semántica nos informa que contiene su estructura general, ademas con ayuda del CSS Grid podemos divir la página en distintas cuadriculas posicionando así sus diversos elementos de forma flexible y sencilla para el usuario final.

Durante la navegación por esta página veremos y explicaremos con mayor profundidad algunas de la funciones que se aplican en las etiquetas semánticas y el por qué se deben utilizar al momento de crear un documento web y su combinación con el Css Grid, dando así solución a lo estipulado en la guia.

Introducción

El lenguaje HTML ha incorporado numerosas etiquetas que podemos clasificar en función de su utilidad o características, Una de ellas son las etiquetas semánticas, las cuales nos indican qué es el contenido que contienen, en lugar de cómo se debe formatear al mostrar el documento HTML en un cliente web.

Dentro de las etiquetas semánticas también encontramos varias funciones que sirven para definir el esquema principal del documento, así nuestra página web mostrara un contenido más util y procesable con mayor facilidad por cualquier sistema informático.
